# simple_api

Sets up Simple API service.

> [!NOTE]
> A simple REST API with Redis database built with Spring boot to learn Docker and Kubernetes.<br>
> URL: https://hub.docker.com/r/jkaninda/simple-api

## Variables

* `simple_api_docker_user`: The user that Docker will run as. This user has to exist already.
* `simple_api_port`: The port that the API is using.
